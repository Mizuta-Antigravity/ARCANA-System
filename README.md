# ARCANA System

ARCANA Systemは、Persona 3 Reloadのビジュアルスタイルを取り入れた、探究意思決定支援システムです。
5つのアルカナ（AI）が、あなたの提案を多角的に分析し、審議を行います。

## 特徴
- **P3RスタイルUI**: 未来的なネオンテーマ、3Dフリップ演出、スキャンライン効果。
- **5つのアルカナ分析**: Magician, Justice, Lovers, Empress, Starの5人格が独自の基準でスコア化。
- **動的サウンド**: Web Audio APIを使用した、没入感のあるサウンドフィードバック。
- **ドキュメント抽出**: PDFやテキストファイルからの内容抽出（プレースホルダ機能）。

## 構成
- `index.html`: メイン構造
- `style.css`: スタイリング（P3Rテーマ）
- `magi.js`: AIロジック及びオーディオエンジン
- `assets/`: 最適化された画像アセット

## セットアップ
1. このリポジトリをダウンロードまたはクローン。
2. `index.html` をブラウザで開くだけで動作します（サーバー不要）。
