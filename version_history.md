# ARCANA System - バージョン履歴

## 【v5.5.0】 - 2026-03-15（最新）
- **カード演出の再構成**：
    - 3Dフリップ・アニメーションを実装。応答時にカードが裏返り、詳細な解析スコアを表示するように改善。
    - パーティクル・エフェクトとカードの重なり（z-index）を連動させ、解析中のアルカナが最前面に表示されるよう調整。
- **音響システムのブラッシュアップ**：
    - P3R（Persona 3 Reload）風のハイブリッド・サウンドセット（No.02）を追加。
    - カードの回転、停止、展開の各フェーズに固有のSEを割り当て。
- **UIスタックの最適化**：
    - モニター解像度に応じたレスポンシブなカード配置（リニア・ファン形式）の計算式を修正。
    - 4K環境下でのテクスチャ解像度のちらつき（エイリアシング）を軽減。

## 【v5.2.8】 - 2026-03-14
- **名称変更**：システム名称を「MAGI」から 「ARCANA System」 に変更。
- **アルカナ人格の導入**：従来のMELCHIOR型科学名称をタロットのアルカナ名にコンバート。
    - MAGICIAN（魔術師）：科学 / 技術的実現性
    - JUSTICE（正義）：倫理 / 社会正義
    - LOVERS（恋人）：人間心理 / 受容性
    - EMPRESS（女帝）：経済 / 生産効率
    - STAR（星）：未来予見 / 持続可能性
- **AIユニット拡張**：AI人格を3体から5体へ増加し、意思決定の多角化を実現。
- **自動分析機能（α）**：ファイルをドロップすると、テキストを自動解析し「具体的な提案内容」へ流し込むデータ抽出エンジンを搭載。

## 【v5.0.0】 - 2026-03-14
- **アーキテクチャ刷新**：
    - 入力画面（Boot/DIVE）と結果画面の2画面分割構造を採用。
    - Web Audio APIによる動的なサウンド合成システムをベースに構築。
- **Lore（設定）オーバーレイ**：各人格の背景情報を閲覧できる「SYSTEM LORE」機能を追加。
- **ビジュアル・テーマ**：ネオン・ブルーとグラスモーフィズム、CRTスキャンライン効果を組み合わせた「Persona-Like」なデザイン言語を確立。

## 【v3.0.0】 - 2026-03-12
- **五角形レイアウトのプロトタイプ**：
    - 3人格から5人格への拡張を見越した、五角形配置（Pentagon Layout）の数学モデルを実装。
- **ファイル読み込み基盤**：PDF.jsおよびJSZipを活用したドキュメント解析の基礎ロジックを統合。

## 【v1.0.0】 - 2026-03-10
- **PROJECT KATANA 発足**：
    - 産学連携および意思決定プロセスの可視化を目的とした「MAGI」プロトタイプ。
    - 3つの基幹AIユニット（MELCHIOR / BALTHASAR / CASPER）によるキーワードマッチング型分析エンジンを初搭載。
