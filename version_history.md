# ARCANA System - バージョン履歴

## 【v5.7.0】 - 2026-03-15（最新）
- **Loreシステムの刷新**：
    - ARCANA人格の解説を詳細化（各人格の専門領域と背景設定を追加）。
    - 解説画面のレイアウトを改善し、テキストの重なりを解消。アイコンとテキストの2カラム構成を採用。
- **モバイル表示の最適化**：
    - Loreモーダルをモバイル時に全画面表示に切り替え、可読性を向上。
    - スキュー（傾斜）エフェクトをモバイル環境で無効化し、レイアウトの安定性を確保。

## 【v5.6.0】 - 2026-03-15
- **レスポンシブ対応の全面強化**：
    - スマートフォン、タブレット、デスクトップの各画面解像度に合わせてレイアウトが動的に変化するアダプティブ・デザインを実装。
    - モバイル表示時にヘッダー要素が重ならないようスタック構造を調整。
- **カードUIの大型化と柔軟性向上**：
    - カード（AIユニット）のベースサイズを拡大し、視認性を向上。
    - 絶対配置からフレックスボックス・ベースの配置へ移行し、画面幅に合わせて自動的に折り返し・拡大縮小を行うように改良。
- **UIバグ修正**：
    - 画面右上の「SYSTEM LORE」ボタンが画面端で切れてしまう問題を修正。
    - 入力パネルのスキュー（傾き）設定が小さい画面で崩れる問題を解消。

## 【v5.5.0】 - 2026-03-15
- **カード演出の再構成**：
    - 3Dフリップ・アニメーションを実装。解析中のアルカナが最前面に表示されるよう調整。
- **音響システムのブラッシュアップ**：
    - P3R風のハイブリッド・サウンドセット（No.02）を追加。

## 【v5.2.8】 - 2026-03-14
- **名称変更**：システム名称を「MAGI」から 「ARCANA System」 に変更。
- **アルカナ人格の導入**：従来のMELCHIOR型科学名称をタロットのアルカナ名にコンバート。
    - MAGICIAN（魔術師）：科学 / 技術的実現性
    - JUSTICE（正義）：倫理 / 社会正義
    - LOVERS（恋人）：人間心理 / 受容性
    - EMPRESS（女帝）：経済 / 生産効率
    - STAR（星）：未来予見 / 持続可能性
- **AIユニット拡張**：AI人格を3体から5体へ増加し、意思決定の多角化を実現。
- **自動分析機能（α）**：ファイルをドロップすると、テキストを自動解析し「具体的な提案内容」へ流し込むデータ抽出エンジンを搭載。

## 【v5.0.0】 - 2026-03-14
- **アーキテクチャ刷新**：
    - 入力画面（Boot/DIVE）と結果画面の2画面分割構造を採用。
    - Web Audio APIによる動的なサウンド合成システムをベースに構築。
- **Lore（設定）オーバーレイ**：各人格の背景情報を閲覧できる「SYSTEM LORE」機能を追加。
- **ビジュアル・テーマ**：ネオン・ブルーとグラスモーフィズム、CRTスキャンライン効果を組み合わせた「Persona-Like」なデザイン言語を確立。

## 【v3.0.0】 - 2026-03-12
- **五角形レイアウトのプロトタイプ**：
    - 3人格から5人格への拡張を見越した、五角形配置（Pentagon Layout）の数学モデルを実装。
- **ファイル読み込み基盤**：PDF.jsおよびJSZipを活用したドキュメント解析の基礎ロジックを統合。

## 【v1.0.0】 - 2026-03-10
- **PROJECT KATANA 発足**：
    - 産学連携および意思決定プロセスの可視化を目的とした「MAGI」プロトタイプ。
    - 3つの基幹AIユニット（MELCHIOR / BALTHASAR / CASPER）によるキーワードマッチング型分析エンジンを初搭載。
